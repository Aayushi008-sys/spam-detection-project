# Spam Detection Project

## Overview
This project detects whether a message is Spam or Not Spam using Machine Learning and Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Features
- Train a spam detection model
- Predict spam messages
- Simple Streamlit web application

## Project Files
- app.py
- model_training.py
- spam.csv
- spam_model.pkl
- vectorizer.pkl
- requirements.txt

## Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Train the model:

```bash
python model_training.py
```

Run the web app:

```bash
python -m streamlit run app.py
```
